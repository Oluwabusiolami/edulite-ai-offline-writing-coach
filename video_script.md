# EduLite AI 2-Minute Demo Video Script

## Opening

Hello, my name is Oluwabusolami Ogunsakin, and my project is called EduLite AI.

EduLite AI is an offline English writing coach designed for African secondary schools.

## The Problem

Many students need help with English writing, grammar, vocabulary and exam preparation.

However, in many African schools, internet access is not always reliable. Some students and teachers cannot depend on cloud-based AI tools every day.

## The Solution

EduLite AI solves this problem by providing a local AI writing assistant that can run offline on an ordinary 8GB laptop.

It supports essay correction, creative writing, vocabulary building, IGCSE ESL writing practice and teacher-style feedback.

## Demo

For example, a student can paste a weak paragraph and ask EduLite AI to improve it.

The AI gives simple feedback, explains the weaknesses and rewrites the paragraph in a better way.

A teacher can also paste a student’s writing and receive suggested comments, grammar corrections and improvement advice.

## African Use Case

This project is useful for schools with limited internet access, computer labs, hostels, libraries and homes.

It helps students practise English even when there is no internet connection.

## Technical Summary

EduLite AI uses a small GGUF language model running locally through llama.cpp.

The goal is to keep it lightweight enough for an 8GB laptop while still giving useful English writing support.

## Closing

EduLite AI does not replace the teacher. It supports teachers and gives students more opportunities to practise.

This project shows how offline AI can make learning support more accessible in African schools.
